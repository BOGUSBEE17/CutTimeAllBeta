📝 Über das Tool

CutTimeAll ist ein simples, aber effizientes Tool zum Schneiden und Zusammenfügen von Videos anhand von Start-/Endmarkierungen.
Entwickelt für einen schnellen, klaren Workflow. Basierend auf dem klassischen time.txt-Konzept.

🔧 Funktionen

  • Start-/Endzeit mit der Leertaste markieren
  • Bearbeitung mehrerer time.txt-Dateien
  • Automatisches Zusammenfügen mit ffmpeg (MP4)
  • VLC-Integration für präzise Videowiedergabe
  • Benutzerdefinierte Pfade & Einstellungen

🚀 Erste Schritte

  1. Installiere VLC: www.videolan.org
  2. Starte CutTime.exe
  3. Video laden, Leertaste drücken – fertig

💡 Tipps

  • Mit dem Edit-Button kannst du Fehler korrigieren
  • Aktiviere Auto-Merge für automatischen MP4-Export
  • Schau dir die Beispielordner zur Orientierung an

📝 About the Tool

CutTimeAll is a simple yet powerful tool to cut and merge video clips using start/end markers.
Designed for fast, clean workflows – based on the classic time.txt method.

🔧 Features

  • Mark start/end points with the spacebar
  • Edit multiple time.txt files
  • Automatically merge clips into MP4 using ffmpeg
  • Integrated VLC player for stable playback
  • Custom paths & flexible settings

🚀 Getting Started

  1. Install VLC: www.videolan.org
  2. Run CutTime.exe
  3. Load a video and press spacebar – done

💡 Tips

  • Use the Edit button to correct timing mistakes
  • Enable auto-merge to generate an MP4 instantly
  • Check the sample folders for orientation
